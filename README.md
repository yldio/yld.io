## How to run

```bash
  git clone git@github.com:yldio/yld.io.git
  cd yld.io
  yarn
  yarn develop
```

In order to access the development branch on contentful during development set the environment variable `ENV` to `'dev'`

```bash
ENV=dev yarn develop
```
